# PartnerNFT


This project aims to help people who can't afford a NFT alone and wants to buy it with a partner or partners.

- First Connect your phantom wallet to the Dapp:

![image](https://user-images.githubusercontent.com/49502963/215188588-a0c1379a-c6a2-4064-9689-7bc5c7039bf4.png)

- Then Create Room specify which NFT do you want to buy and how many people you are going to buy with:

![image](https://user-images.githubusercontent.com/49502963/215190156-5082ea9c-6e0b-46a4-a9f2-99dadea5f1ee.png)

- After creating room you need to share your room id with your partner(s)

![image](https://user-images.githubusercontent.com/49502963/215190258-476ec6c7-7c50-40b8-b29a-0736860599d8.png)

- When every partner approve the buying action Dapp will automatically buy the specified NFT on floor:

![image](https://user-images.githubusercontent.com/49502963/215190367-8bcca439-1663-4a31-82aa-739462acbb62.png)

- Same thing apply for the selling action:

![image](https://user-images.githubusercontent.com/49502963/215190414-55e217c3-9ae5-4b3b-ad0d-7af0403d60c1.png)
